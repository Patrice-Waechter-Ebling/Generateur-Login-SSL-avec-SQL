#### Generateur-Login-SSL-avec-SQL
    Generateur Logging chiffré a partir d'un mot de passe et d'un nom d'utilisateur demandé a l'utilisateur 
    en utilisant un service de cryptographie et un salt en  c++ console
    une version avec déchiffrement intégré
    une version utilisant un algorithme plus fort (MCE2026 maison)
    une version avec stockage dans une database SQL
    generation du script SQL approprie pour la table obtenue
#### SQL
    un schéma MS‑SQL propre
    une procédure d’insertion
    une procédure de lecture
    les types recommandés pour stocker du sel, du ciphertext et l’algorithme
    un appel C++ (ODBC)
